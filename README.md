# Storyscript Library

## Project setup

```
yarn
```

### Run styleguide

```
yarn serve
```

### Compiles and minifies the library for production

```
yarn build:lib
```

### Compiles the styleguide for production

```
yarn build:styleguide
```

### Lints and fixes files

```
yarn lint
```

### Run the unit tests

```
yarn test:unit
```
