### Font sizes

```js
<div>
  <h1 class="is-size-1">I'm a title size 1</h1>
  <h2 class="is-size-2">I'm a title size 2</h2>
  <h3 class="is-size-3">I'm a title size 3</h3>
  <h4 class="is-size-4">I'm a title size 4</h4>
  <h5 class="is-size-5">I'm a title size 5</h5>
  <p class="is-size-6">I'm a paragraph size 6</p>
  <p class="is-size-7">I'm a paragraph size 7</p>
</div>
```

### Text helpers

```js
<div>
  <h1 class="is-size-3 has-text-left">I'm a left-centered text</h1>
  <h1 class="is-size-3 has-text-centered">I'm a middle-centered text</h1>
  <h1 class="is-size-3 has-text-right">I'm a right-centered text</h1>
  <h1 class="is-size-3 has-text-uppercase">I'm an uppercased text</h1>
</div>
```
